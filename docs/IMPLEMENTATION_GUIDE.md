# Guía de Implementación

## 1. Implementa las interfaces públicas

- UniversalObject
- UniversalScreen
- UniversalViewModel
- UniversalFocusView

## 2. Carga y parsea los JSON de ejemplo

- Usa cualquier lenguaje compatible con JSON
- Instancia los objetos según el contrato

## 3. Renderiza y navega según los datos

- No codifiques lógica de dominio
- Aplica efectos y visuales según los campos declarativos

## 4. Extiende para nuevos dominios

- Solo agrega nuevos JSON, no cambies el motor
